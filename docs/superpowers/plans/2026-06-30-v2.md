# v2.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver three modules — ApiAccessLog 日志优化, SSO 登录改造, 技改任务管理 — following the design spec at `docs/superpowers/specs/2026-06-30-v2-design.md`.

**Architecture:** P0 (ApiAccessLog) touches one AOP aspect; P1 (SSO) adds a user-info service mock, new AuthController endpoints, and frontend AES-cached userInfo; P2 (技改) introduces a parent→subtask→item table hierarchy, backend CRUD in `techmg-business-service`, and two Vue pages (overview + dual-tab subtask).

**Tech Stack:** Spring Boot 3.2.3, MyBatis-Plus 3.5.5, MySQL 8, jjwt 0.12.3, Vue 3.3.9, Element Plus 2.5.0, Apache POI 4.1.2, Gson 2.10.1

## Global Constraints

- Java 17+, package `com.icbc.sh.techmg`
- Config `.yml` only (no `.properties`)
- All APIs return `R<T>` (code/message/data)
- MyBatis XML for complex SQL; no `@Select/@Update/@Insert/@Delete`
- JSON via Gson, `extendMessageConverters` must preserve String converter
- Log4j2 2.17.1, exclude Logback
- Spring Security stateless JWT, `/api/auth/**` public
- `@ApiAccessLog` on new controller endpoints
- Logical delete (`deleted`), `@TableLogic`
- Auto-fill via `MyMetaObjectHandler` for `createTime/createBy/updateTime/updateBy`
- Business exceptions via `BusinessException` → `GlobalExceptionHandler`

- [ ] **Step 3: Add SSO login endpoint to AuthController**

Inject `SsoAuthProvider` (using `@Autowired(required = false)` so it's optional):

```java
import com.icbc.sh.techmg.framework.security.SsoAuthProvider;
import org.springframework.beans.factory.annotation.Autowired;

// Add field:
@Autowired(required = false)
private SsoAuthProvider ssoAuthProvider;

// Add after existing login() method:

@ApiAccessLog
@PostMapping("/sso/login")
public R<Map<String, Object>> ssoLogin(@RequestBody Map<String, String> request) {
    if (ssoAuthProvider == null || !ssoAuthProvider.enabled()) {
        return R.fail(400, "SSO is not enabled");
    }

    String ticket = request.get("ticket");
    if (ticket == null || ticket.isBlank()) {
        return R.fail(400, "Missing SSO ticket");
    }

    // Step 1: Validate SSO ticket → get userId (authNo)
    String authNo = ssoAuthProvider.authenticate(ticket);
    if (authNo == null || authNo.isBlank()) {
        return R.fail(401, "SSO authentication failed");
    }

    // Step 2: Query user extended info from external API (TODO → mock for now)
    Map<String, Object> extInfo = ssoAuthProvider.getUserInfo(authNo);
    if (extInfo == null) {
        extInfo = new HashMap<>();
        extInfo.put("authNo", authNo);
    }

    // Step 3: Sync to sys_user (upsert + update last_login_time)
    SysUser sysUser = sysUserService.syncUserInfo(extInfo);

    // Step 4: Load roles and generate JWT
    SysUser finalUser = sysUserService.getByAuthNo(authNo);
    List<String> roles = loadRolesForUser(finalUser);

    String token = jwtTokenProvider.generateToken(authNo,
        roles.stream().map(org.springframework.security.core.authority.SimpleGrantedAuthority::new)
             .collect(Collectors.toList()));

    // Step 5: Build response with extended user info
    Map<String, Object> userInfo = new LinkedHashMap<>();
    userInfo.put("authNo", authNo);
    userInfo.put("realName", finalUser.getRealName());
    userInfo.put("tellername", extInfo.getOrDefault("tellername", ""));
    userInfo.put("ad", extInfo.getOrDefault("ad", ""));
    userInfo.put("branchId", extInfo.getOrDefault("branchId", ""));
    userInfo.put("branchName", extInfo.getOrDefault("branchName", ""));
    userInfo.put("notesId", extInfo.getOrDefault("notesId", ""));
    userInfo.put("branchIdList", extInfo.getOrDefault("branchIdList", Collections.emptyList()));
    userInfo.put("roles", roles);

    Map<String, Object> result = new HashMap<>();
    result.put("token", token);
    result.put("userInfo", userInfo);
    return R.ok(result);
}

private List<String> loadRolesForUser(SysUser sysUser) {
    if (sysUser == null) return List.of();
    return sysUserService.getRoles(sysUser.getId()).stream()
        .map(role -> "ROLE_" + role.getRoleCode())
        .collect(Collectors.toList());
```

```javascript
const SECRET_KEY = import.meta.env.VITE_USERINFO_ENCRYPT_KEY || 'techmg-default-key'

export function encrypt(data) {
  const json = JSON.stringify(data)
  return CryptoJS.AES.encrypt(json, SECRET_KEY).toString()
}

export function decrypt(ciphertext) {
  try {
    const bytes = CryptoJS.AES.decrypt(ciphertext, SECRET_KEY)
    const json = bytes.toString(CryptoJS.enc.Utf8)
    return JSON.parse(json)
  } catch (e) {
    console.warn('Failed to decrypt userInfo:', e.message)
    return null
  }
}
```

```java
public ParseResult parseFile(MultipartFile file) {
    String filename = file.getOriginalFilename();
    String ext = filename != null ? filename.substring(filename.lastIndexOf(".") + 1).toLowerCase() : "";

    List<String[]> rawData;
    switch (ext) {
        case "csv":
        case "txt":
            rawData = parseCsv(file);
            break;
        case "xls":
        case "xlsx":
            rawData = parseExcel(file);
            break;
        default:
            throw new BusinessException("不支持的文件格式: " + ext);

    if (rawData.isEmpty()) {
```
